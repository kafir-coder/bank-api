## How to execute

first you need to clone the repo
```bash
git clone git@github.com:kafir-coder/bank-api.git
```
now you can execute the API just typing
```bash
docker-compose up
```
## Testing the API

```python
# without coverage informations
npm run test

# with coverage informations
npm run test:ci

```
